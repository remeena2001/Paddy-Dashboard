# Paddy Dashboard
An interactive Paddy Production Analytics Dashboard designed to uncover actionable insights into Sri Lanka's agricultural landscape.
🛠️ Technical Skills & Tools Used
Tool: Power BI Desktop

Data Preparation: Cleaned and shaped raw agricultural .csv data using Power Query.

Data Modeling: Built a structured model to connect seasonal timelines with district-level metrics.

DAX (Data Analysis Expressions): Authored custom measures to extract advanced analytical insights, including:

Crop Failure Area (Calculating the exact hectare loss)

Total Production summaries

Data Storytelling: Focused on UI/UX best practices (clean titles, intentional color-coding for success vs. warning metrics, and hiding unnecessary labels) to ensure the dashboard is readable for non-technical stakeholders.
